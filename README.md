# Bank Loan Data Analysis – Python Project

## Overview

This project analyzes a **bank loan dataset** using Python to identify trends, performance metrics, and borrower characteristics.
The analysis includes **data loading, data cleaning, exploratory data analysis (EDA), and visualization** to understand loan performance, funding trends, and borrower attributes.

The final insights were summarized and presented using **Gamma AI to generate a presentation (PPT)** for clear communication of results.


## Dataset

The dataset used in this project contains **bank loan information**, including borrower details, loan amounts, loan status, interest rates, and other financial attributes.

Key fields analyzed include:

* Loan amount
* Issue date
* Loan purpose
* Loan term
* Employment length
* Home ownership
* Interest rate
* Debt-to-income ratio
* Loan status

These variables were used to calculate metrics such as funded amount, loan performance, and borrower trends.

## Project Steps

### 1. Data Loading

* Imported required Python libraries.
* Loaded the **Financial Loan dataset** from a local Excel file into Jupyter Notebook.

### 2. Data Understanding

* Examined dataset structure and metadata.
* Checked data types, columns, and basic dataset statistics.

### 3. Data Cleaning

* Handled missing values where necessary.
* Verified column data types.
* Prepared variables for analysis.

### 4. Exploratory Data Analysis (EDA)

Several business metrics and analyses were performed, including:

**Loan Performance Metrics**

* Total Loan Applications
* Month-to-Date (MTD) Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Debt-to-Income Ratio (DTI)

**Loan Quality Metrics**

* Good Loan Metrics
* Bad Loan Metrics

**Trend Analysis**

* Monthly trends by issue date for:

  * Total funded amount
  * Total amount received
  * Total loan applications

**Borrower & Loan Insights**

* Regional analysis by state
* Loan term analysis
* Employment length analysis
* Loan purpose analysis
* Home ownership analysis

Various charts such as **line charts, bar charts, donut charts, and treemaps** were created to visualize insights.

## Results / Insights

The analysis helped identify:

* Trends in **loan funding and repayments over time**
* Differences in loan funding across **states**
* Loan distribution by **term and purpose**
* Borrower characteristics such as **employment length and home ownership**
* Key indicators of **good vs bad loan performance**

These insights can help financial institutions better understand lending patterns and borrower risk.
